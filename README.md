## Project Information
The project is an audit of open source software. The audit involves the origin of the software and open source license. The project also involves implementing Linux shell scripting with the development of 5 shell scripts.  

---

## Overview of Scripts
### Script 1: System Identity Report
This script reports the details of the system. The system details include user name and kernel version. The system up time is also reported by this script. The date is also reported.  

### Script 2: FOSS Package Inspector
This script checks whether the Git package is already installed or not. The package is inspected. The package details are reported.  

### Script 3: Disk and Permission Auditor
This script reports the permissions and sizes of the important directories.  

### Script 4: Log File Analyzer
This script analyzes the log files. The number of times the keyword is used is reported.  

### Script 5: Open Source Manifesto Generator
This script generates the open source manifesto.  

---

## How to Run Scripts
### Step 1: Run the Scripts
Open the Ubuntu terminal.  
Change directory to project folder.  
Change permission of the scripts.  
chmod +x script1.sh script2.sh script3.sh script4.sh script5.sh  
Run scripts.  
./script1.sh  
./script2.sh  
./script3.sh  
./script4.sh /var/log/dpkg.log  
./script5.sh
